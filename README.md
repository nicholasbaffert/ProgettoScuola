# 🎰 Casino Console – Mini Gambling Games

## 📌 Descrizione
**Casino Console** è un progetto Python che simula un piccolo casinò da terminale.  
L’utente può scegliere tra diversi mini-giochi, effettuare puntate e gestire un saldo virtuale.

Il progetto è pensato per esercitarsi con:
- logica di programmazione
- gestione input/output
- numeri casuali
- organizzazione modulare del codice

---

## 🎯 Funzionalità

- Menu principale interattivo
- Sistema di saldo virtuale
- Giochi disponibili:
  - 🎡 Roulette
  - 🎲 Dadi
  - 🃏 Blackjack
- Sistema di puntate e vincite
- Controllo input utente
- Aggiornamento saldo in tempo reale
- Possibilità di uscire o continuare a giocare
